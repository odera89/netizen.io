Introduction

Social Auth is an easy configurable, fully dynamic, higly customizable wordpress plugin/widget that lets you authorize your sign-in over well-known networks like facebook, yahoo, google, etc... Script usage is very easy, you don't need to write any code. You can fully integrate this script by just only configuration.

Available Features

  • Login with Facebook
  • Login with Twitter
  • Login with Google
  • Login with Linkedin
  • Login with Yahoo
  • Enable/Disable login buttons
  • Plugin usage
  • Widget usage

System Requirements

  • Wordpress 3.0+
  • PHP 5.0+
  • Applications that are created on Facebook, Twitter, and Linkedin

Installation

  1. Download SocialAuth plugin from Codecanyon
  2. Install it to your wordpress system(Please refer here for more information on Wordpress plugin installation)
  3. Activate plugin from "Plugins" section
  4. Go to SocialAuth menu under "Settings" section.Configure your Social Network credentials.
  5. Use it as plugin by adding this script "<?php echo get_socialauth_login_area(); ?>"  to anywhere you want in theme, or you can use plugin by drag and drop in widget sections.

Backend

After you have installed and activated the plugin, you will see a section called "SocialAuth" in "Settings". Go and configure your plugin!

More on Backend

  • If you do not enter logo url of any social network, it won't be shown in frontend
  • Callback url of social networks is given on information text for each callback input.

Frontend

The ones you activated on backend will be displayed as login button on frontend.
